[English Description]
ModernArchiveThumbnail is a Windows Shell Extension that generates fast and accurate thumbnails for archive files. The legacy C#-based engine (v3.0.0) has been deprecated and completely rebuilt from scratch in Native C++ for ultimate performance and stability.

âœ¨ Key Features
High-Performance Engine: Built with C++ and Native Win32 APIs for minimal resource usage and near-instant response.

Smart Cover Selection: Uses numerical sorting and keyword analysis (cover, front, etc.) to intelligently select the best cover image.

Broad Format Support: Full support for various archive formats including Zip, RAR, 7z, TAR, LZH, ALZ, EGG, and more.

Modern Image Rendering: Perfectly renders modern image formats like WebP, HEIC, AVIF, PNG, and JPG using WIC.

Rock-Solid Stability: Optimized single-thread design ensures zero explorer lag and prevented memory leaks.
