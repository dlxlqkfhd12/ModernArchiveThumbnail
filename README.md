\# Modern Archive Thumbnail Handler



This project provides a shell thumbnail extension for Windows Explorer.

It adds thumbnail support for archive formats and modern image formats:



\- Zip, RAR, CBZ, CBR archives

\- WebP, TIFF, GIF, HEIC / HEIF images



This repository contains:

\- Shell extension source code

\- Installation / uninstallation batch files

\- Inno Setup installer script
